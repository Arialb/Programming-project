Human Genome Specification

In this project, you can further understand the contents of the human genomic data which is in a GFF3 file format; that is a default format when we are studying a genomic data from any species (especially the human). In this page you can see different buttons which corresponds to different functions. These function can bring you to a specific information you're interested in knowing regarding the human genome.

You can see below a list of active operations which will bring you to a new page which contains the infromation that the operation is describing

 Choose a command of your choice:
  1. Get Basic information of each column from the dataset



  2. Get a list of all unique sequence IDs which can be found on the dataset



  3. Obtain a list of all the unique types of the human genome



  4. Count the number of features of the same source



  5. Count the number of entries for each type of operations



  6. Obtain new dataset containing information of entire Chromosome (GRCh38)



  7. Calculating unassembled sequences from the GRCh38



  8. Provides new dataset from ther source



  9. Counts the number of entries for each type of operation of the dataset containing from source



  10. Returns Gene names from the dataset
